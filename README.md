# Full Stack Task Manager Application

This is a simple **Full Stack Task Manager Application** built using a separate **frontend and backend architecture**.  
The backend exposes REST APIs and the frontend interacts with those APIs to manage tasks.

The application demonstrates how a frontend communicates with a backend service which uses an ORM to interact with a cloud database.

---

## Technologies Used

### Frontend
- HTML
- JavaScript

### Backend
- Flask (Python Web Framework)
- SQLAlchemy ORM

### Database
- PostgreSQL (Hosted on Supabase)
